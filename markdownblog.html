Markdown Blog

A small project made with the intent of practicing creating an express server, connecting to MongoDB and have my first interaction with bootstrap via CDN, where I can just include bootstraps compiled CSS/JS. The markdown blog allows users to post, edit and delete articles. The projects makes use of dompurify which sanitizes the HTML that the end users input. This means it will output clean HTML and prevent XSS attacks even if the user enters in strings of messy HTML.
