Crypto Market

This is a project that can display the Top 100 cryptocurrencies in the world, it also shows all the latest cryptocurrency news with the time and source it is from. The project makes use of two separate API from RapidAPI. They are Coin Ranking API and Bing news API. Project is created using ReactJS and Ant design. Ant Design is a popular JavaScript framework component developed by Alibaba. The APIs used are from a free subscription that allows me to make certain amount of requests per month.
