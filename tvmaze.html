TV Maze

This is a simple project where I make use of the API provided by TV Maze. The user can search for any TV show and it will display the results and summary of the chosen show. We fetch data from the TV Maze API based on a searchTerm variable. The fetch function sends a GET request to the API with the search term as a query parameter. With this mini project, we fetch data from an external API and updates the state of a React component based on the retrieved data, allowing for dynamic rendering of the component based on user input.
